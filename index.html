Connect to GitHub

Estimates are stored as a JSON file in your GitHub repo — shared across your whole team. You only need to do this once; your token is saved locally in your browser.

Generate one at github.com/settings/tokens/new. Select the repo scope (classic token) or Contents: read & write (fine-grained).
The file will be created automatically on first save if it doesn't exist yet.
